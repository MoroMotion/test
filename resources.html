Frequently Asked Questions

What programming languages can I use with GLENN? +
GLENN supports Python (primary), C++, and JavaScript through our official SDKs. We also provide ROS2 integration packages for robotics developers.
How often is GLENN's software updated? +
We release quarterly major updates with new features and improvements. Security patches and bug fixes are released monthly. All updates are backward compatible.
Is training required to operate GLENN? +
Basic operations can be learned in a few hours using our Quick Start Guide. Advanced features and mission planning typically require 2-3 days of training, which we provide to all customers.
What support options are available? +
We offer 24/7 technical support via email and phone, quarterly maintenance visits, and access to our customer portal with documentation and community forums.
Can GLENN operate in extreme environments? +
Yes, GLENN is designed for extreme conditions including temperatures from -10¬∞C to 45¬∞C, dust, rain, and rough terrain. For specialized environments, custom configurations are available.
